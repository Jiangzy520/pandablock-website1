🛠️ 如果邮件通知失败，请按以下步骤操作：

  1. 检查 Vercel 环境变量
    访问 Vercel Dashboard → 项目设置 → Environment Variables
    确认 RESEND_API_KEY 已配置
  2. 获取 Resend API Key
    访问 Resend.com → 登录 → API Keys → 创建新 Key
    复制 API Key（格式：re_xxxxxxxxxx）
  3. 在 Vercel 添加环境变量
    Name: RESEND_API_KEY
    Value: 您的 Resend API Key
    Environment: 全选（Production, Preview, Development）
  4. 重新部署项目
    Vercel Dashboard → Deployments → 最新部署 → Redeploy
  5. 再次测试
    返回此页面，点击"发送测试邮件"按钮
